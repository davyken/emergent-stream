# 🎬 Emerging-Stream — Site Web Professionnel

Un tunnel de vente complet et professionnel pour votre service de streaming.

## 🚀 Démarrage rapide

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm run dev

# Build de production
npm run build
```

## 📦 Stack Technique

- **React 18** + Vite
- **Lucide React** (icônes)
- CSS variables (design system)
- Polices Google : Cormorant Garamond + DM Sans

## 🗂️ Structure des sections

| Section | Description |
|---------|-------------|
| `Navbar` | Navigation fixe avec scroll effect |
| `Hero` | Accroche visuelle + stats |
| `LeadMagnet` | Formulaire capture email/WhatsApp |
| `HowItWorks` | Tunnel en 4 étapes |
| `Features` | 8 fonctionnalités avec hover |
| `Pricing` | 3 plans + Order Bump + Upsell + Downsell |
| `Testimonials` | 6 avis clients + trust bar |
| `FAQ` | 8 questions accordéon |
| `CTABanner` | Bannière de conversion finale |
| `Footer` | Liens + réseaux sociaux |

## 🎨 Design

- Thème sombre luxueux (or & teal)
- Animations CSS (fadeUp, shimmer, float, particles)
- Totalement responsive (mobile-first)
- Micro-interactions sur hover

## 📱 Responsive

Optimisé pour mobile, tablette et desktop.
